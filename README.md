# TrustAuth

TrustAuth is a cybersecurity project that detects Banking Account Takeover (ATO) using continuous behavioral authentication through keystroke dynamics.

## Features

- Secure login system
- Continuous authentication
- Keystroke dynamics analysis
- One-Class SVM anomaly detection
- OTP-based step-up authentication
- Security event logging
- Behavioral risk dashboard

## Tech Stack

- Flask
- SQLite
- JavaScript
- Python
- Scikit-learn
- HTML/CSS
